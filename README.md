# NFT.com backend

## Getting Started

### 1. Requirements

1. Node.js version >=16
2. Docker

### 2. Build and Start

1. `npm install`
2. `cp .env.sample .env`; modify this file if needs be
3. `docker-compose up -d`
4. `npm run dev`

Server will start at `localhost:10010`.

Admin portal of local Mongo is available at `localhost:10070`.
