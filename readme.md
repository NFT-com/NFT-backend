# NFT.com backend

## Getting Started

### 1. Requirements

1. Node.js version 14.18.1
2. Docker
3. Yarn

### 2. Build and Start

1. `yarn install`
2. `cp .env.sample .env`; modify this file if needs be
3. `docker-compose up -d`
4. `yarn dev`

Server will start at `localhost:10080`.

Admin portal of local Mongo is available at `localhost:10070`.
